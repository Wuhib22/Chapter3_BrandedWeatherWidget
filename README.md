Branded Weather Widget – JavaFX

A desktop weather widget built in JavaFX, designed to reflect the style and personality of a company. Shows the current weather, a 3-day forecast, and extra features tailored to the company’s users.

EcoLife Solutions: Gardening tips, air quality, water-saving suggestions

Urban Pulse Labs: Commute impact, UV index, public transport info
